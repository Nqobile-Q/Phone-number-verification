# Phone Number Verification in Zoho CRM

This project is a **Phone Number Verification system** built using **Deluge scripting** in **Zoho CRM**. It validates mobile numbers to ensure they are in the correct format, helping reduce errors in data entry and improve CRM data quality.

## Features
- Validates phone numbers for correct format.
- Sends error messages for invalid entries.
- Integrates directly within Zoho CRM forms for real-time verification.

## What I Learned
- How to write and implement **Deluge scripts** in Zoho CRM.
- Techniques for **data validation** in CRM systems.
- Handling **error messages** and improving **user input accuracy**.
- Basics of **CRM automation** and workflow integration.

## Technologies Used
- Zoho CRM
- Deluge Scripting

## Usage
1. Add the Deluge script to your Zoho CRM form workflow.
2. Set the script to trigger on **record creation or edit**.
3. Test by entering phone numbers to see validation in action.
